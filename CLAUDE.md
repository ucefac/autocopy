# 项目说明

该仓库多平台支持：MacOS

选中文字、双击、三击后模拟按键 Cmd（Ctrl）+C，实现自动复制到剪贴板，无需手动按。 

# Agent 指南

当你设计或者修改时需要考虑多平台支持，根据当前任务决定操作系统相关事宜。

# 核心模块说明

目前已实现的核心功能模块：
1. **ClipboardManager**: 剪贴板管理模块，负责处理系统剪贴板的读写操作
2. **EventListener**: 事件监听模块，负责监听系统级的鼠标选中、双击、三击等用户操作事件
3. **ToastManager**: 提示框管理模块，负责操作成功/失败等状态的 toast 通知展示
4. **StatusBarManager**: 状态栏管理模块，负责在系统状态栏显示应用运行状态、提供快捷操作入口
